# PDF-OCR

A local web-based tool for PDF processing and OCR with LLM-powered bid evaluation.

## Setup

1.  Create a virtual environment:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3.  Run the backend:
    ```bash
    python run.py
    ```
